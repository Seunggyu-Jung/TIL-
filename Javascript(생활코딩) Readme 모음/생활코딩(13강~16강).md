# 생활 코딩 자바스크립트 학습(13~17강): 조건문 기본 문법


## 1). 프로그램, 프로그래밍, 프로그래머

1-1. 프로그램 : 작업의 순서
1-2. 프로그래밍: 작업의 순서를 배치하는 작업
1-3. 프로그래머: 그 순서를 배치하는 사람

## 2). 조건문

2-1. 비교 연산자와 불리언

- 데이터 타입의 종류: <br>
(1).숫자(number) <br>
(2). 언어(string) <br>
(3). Boolean: True, False

- 비교 연산자: === -> 좌항과 우항이 같으면 True, 다르면 False를 출력 <br>

- 조건문의 형식:

- ```<br>```: 줄바꿈
